# 密码泄露渠道

  1. 数据库被偷
  2. 服务器被入侵
  3. 通讯被窃听
  4. 内部人员泄露数据
  5. 其他网站(撞库)

# 解决方案

  1. 严禁明文存储密码
  2. 单项变换（只能是明文到密文 不能从密码转明文）
  3. 变换复杂度要求 （使用加密算法加密）
  4. 密码复杂度要求（用户注册 密码有要求）
  5. 加盐

# 哈希算法

# 特点

  • 明文-密文一一对应
  • 密文-明文无法反推
  • 密文固定长度
  • 雪崩效应 （修改一点点明文 密文完全不一样）

# 常见哈希算法

  • md5
  • sha1
  • sha256
