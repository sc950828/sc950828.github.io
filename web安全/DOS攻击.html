# 定义

DOS（拒绝服务攻击），模拟大量虚拟用户请求，大量占用服务器资源。

DDOS（大规模分布式拒绝服务攻击） 攻击的是大量分布式的请求过来，请求量远大于 DOS 攻击方式

# 攻击方式

TCP 半连接

HTTP 连接

DNS， 攻击 DNS 服务器

# 防御方式

防火墙

交换机、路由器

流量清洗

高防 IP（云服务厂商一般有这个服务）
