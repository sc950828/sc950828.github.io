## 定义

通过用户的行为完成某操作，但是用户并不知情。

## 注入方式

主要是通过嵌入 iframe 到网站中并设置 opticy 为 0，再诱导用户进行一系列操作。

## 防御

### 使用 js 禁止内嵌 iframe

```js
if (top.location != window.location) {
  top.location = window.location;
}
```

### 禁止内嵌

设置响应头`X-Frame-Options`

- DENY 禁止任何内嵌 iframe
- ALLOW-FROM 指明指定的域可以内嵌 iframe
- SAMEORIGIN 同域才能内嵌 iframe
